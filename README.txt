== Description ==

This plugin allows you to add comments manually. Useful to migrate comments from a Facebook Fan Page to your website.

Suppose you have a Facebook Fan page. Many people send you messages there but only a few do it in your site. To solve this you decide to copy the comments and private messages from Facebook to Wordpress. This is where this plugin can help you.

### How to use it

- Copy comments from you Facebook Page into Wordpress
- Write the name of the user
- Select the post where you want it to be placed
- Done



== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Copito Comments, login to your WordPress dashboard, navigate to the Plugins menu and click Add New.

1. In the search field type Copito Comments and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

2. Activate the plugin in the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard

2. Navigate to the 'Upload' area

3. Select `copito-comments.zip` from your computer

4. Click 'Install Now'

5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `copito-comments.zip`

2. Extract the `copito-comments` directory to your computer

3. Upload the `copito-comments` directory to the `/wp-content/plugins/` directory

4. Activate the plugin in the Plugin dashboard



== Frequently Asked Questions ==

= What is this plugin used for? =

Most of your customers usually contact you via social media. This plugin allows you to migrate those comments to your site.

= How do I use it? =

Open the menu called **Copito Comments**, fill in the form with a name, a comment and select the post where you want it to be posted.

= What else can I do with this plugin? =

You could create fake comments with it.
